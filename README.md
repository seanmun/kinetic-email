## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/kinetic-email.git
   cd kinetic-email
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   ```
   http://localhost:5173
   ```

### Available Scripts

```bash
# Development
npm run dev          # Start development server
npm run build        # Build for production
npm run preview      # Preview production build
npm run lint         # Run ESLint

# Deployment

```

## ğŸ”§ Development Guidelines

### Code Style
- **TypeScript** for type safety
- **Tailwind CSS** for consistent styling
- **Modular components** with single responsibility
- **React Icons** instead of image assets

### File Organization
- Group related components in subdirectories
- Use relative imports for local components
- Separate data from UI components
- Keep examples in dedicated files

### Performance
- Lazy load content for larger pages
- Optimize images and assets
- Avoid complex CSS selectors
